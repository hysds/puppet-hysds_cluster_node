# HySDS Cluster Node

Puppet module to install all software for a generic HySDS cluster node.


## Prerequisites
Create a base CentOS7 image as described [here](https://github.com/hysds/hysds-framework/wiki/Puppet-Automation#create-a-base-centos-7-image-for-installation-of-all-hysds-component-instances).


## Installation
As _root_ run:
```
bash < <(curl -skL https://github.com/hysds/puppet-hysds_cluster_node/raw/master/install.sh)
```
